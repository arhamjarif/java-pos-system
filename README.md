# Java POS System 

**FEATURES**  
- Java swing Graphical User Interface  
- Browse available products and corresponding prices  
- Select quantities of products for order  
- Automatic calculation of total cost  
- Customer order placement  
- Save each order to "order.txt" using Java file I/O

A java project that shows a simple ordering window where you may see available store item and their price. 
You may select how many of each items you wish to order, see the total price and order.
Upon ordering the program will make a note of the incoming order in the orders.txt file.
