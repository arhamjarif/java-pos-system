# Java POS System

A simple Point of Sale (POS) application built using Java Swing that allows users to browse products, place customer orders, calculate totals, and save orders to a text file.

## Features

- Java Swing graphical user interface
- Product catalogue with prices
- Quantity selection
- Automatic total price calculation
- Order placement
- Save customer orders to a text file

## Concepts Implemented

- Java Swing
- Event-driven programming
- Object-oriented programming
- File I/O
- GUI design

## Requirements

- Java JDK

## How to Run

1. Compile the Java source files.
2. Run `Start.java`.
3. Select product quantities.
4. Review the total price.
5. Place the order.

## Future Improvements

- Inventory management
- Order history
- Database integration
